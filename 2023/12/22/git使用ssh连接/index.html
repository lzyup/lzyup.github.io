Git使用ssh连接

分类于 工具 ， Git

不存在 SSH 秘钥

  • 打开 github->setting->sshkeys and GPGkeys
  • 输入 ssh-keygen -t rsa -C “xxxx@xxx.com“(你的邮箱)
  • cat ~/.ssh/id_rsa.pub
  • 复制产生的 ssh keys，并粘贴到 github 对应页面位置中

ssh keys 过期失效

  • 检查 SSH keys 是否已经存在
    • ls ~/.ssh/
  • 将获取的 SSH Keys 添加到 id_rsa
    • ssh-agent bash
    • ssh-add ~/.ssh/id_rsa
  • 测试是否链接成功
    • ssh -T git@github.com
